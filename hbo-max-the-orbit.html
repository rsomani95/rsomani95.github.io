HBO Max - "The Orbit"

Special Circumstances Ã— HUSH - 2021

Worked on building automated systems that selected diverse, relevant, and important scenes from the HBO archive that were bubbled up to the interactive front end. My specific contributions were:

  • Created a custom architecture with a shared backbone that jointly optimised for all the tasks. Having a shared backbone not only reduced inference costs, but also led to superior performance than individual models for all the tasks due to the shared representation created by the joint optimisation
  • Wrote custom training loops to fine-tune the model efficiently. Tricks used included gradual unfreezing, gradient accumulation, custom data augmentation, etc.
  • Improved our in-house datasets for recognising lighting, color, framing, location and composition cues more robustly
  • Built a custom NSFW classification dataset tailored to the HBO archive
  • Heuristics on top of model outputs to understand edit structure, build a robust set of criteria to filter out irrelevant scenes
  • Helped brainstorm with the larger team on models and tools for downstream analysis of dialogue and facial keypoints
  • Optimised model with ONNXRuntime to analyse near petabyte scale archive run on AWS
